Avantage

Les Avantages des application pour les daltonyen

Les Application permettent de d'avoir des couleurs adapté pour le daltonyen rapidement et peux être nativement activable

Desavantage

Les desavantage des application pour les daltonyen

Les Application peuvent nécéssiter une instalation de logiciel qui peut être long et ne fourni qu'une aide limité. Si l'utilisateur n'est pas un G@merZ il peut éprouver des difficulté a adapté l'application a son propre type de dyschromatopsie.

Microsoft sort une app pour aider les daltoniens à distinguer les couleurs Julien Cadot 18 novembre 2016 - Tech

Accueil Tech Microsoft sort une app pour aider les daltoniens à distinguer les couleurs Color Binoculars est une application conçue par Microsoft pour aider les daltoniens à mieux distinguer les couleurs et les formes. Nous l'avons testée. Microsoft vient de sortir sur l’App Store américain une application fort intéressante, nommée Color Binoculars : il s’agit d’une caméra qui souhaite aider les daltoniens à mieux distinguer les couleurs. Côté interface, Redmond a fait dans le simplissime : l’application ne présente qu’un bouton on / off et trois options pour trois différents types de daltonismes : Rouge/Vert, Vert/Rouge et Bleu/Jaune.

Une extension Chrome pour les daltoniens

« Colorblindly » est justement une extension Chrome qui vous permet de vous mettre dans la peau d’un daltonien et de voir ainsi comment il perçoit le web.

Ce léger handicap peut être contourné en appliquant un algorithme spécifique pour remplacer dynamiquement le vert et le rouge par d’autres couleurs plus adaptées. Il existe même un outil, Color Oracle, qui permet de simuler la vision d’un daltonien pour adapter au mieux le design d’un site web. Mais trop peu d’éditeur s’attache à ce « détail ».

Pragmatique, Animesh propose donc d’adresser le problème directement dans le navigateur, sans tenir compte de la capacité des entreprises à améliorer le niveau d’accessibilité de leur site. Il développe donc actuellement une extension pour Chrome, le navigateur de Google. Cette dernière permettra à l’internaute de choisir et régler les paramètres en fonction de sa vision.

L’initiative a séduit des géants comme Google et Intel qui l’ont invité à des évènements tels que l’Intel International Science and Engineering Fair et le Google Science Fair. Le projet est aussi financé en partie via la plate-forme collaborative Indiegogo.

Encore une démonstration de l’intérêt croissant des développeurs et des géants de l’informatique pour l’écoconception logicielle. Sources : http://www.animesh.co/cb/, http://animesh.co/scifairprogram/, https://student.societyforscience.org/intel-isef
